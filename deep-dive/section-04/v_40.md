## truncation

## Floor

x = 10.4

the floor is 10

What about `x = -10.4

the floor is -11

```
import math
math.floor(10.4)
10

math.floor(-10.4)
-11
```

## Ceiling

The opositive of floor

x = 10.4

the ceiling is 11

x = -10.4

the ceiling is -10

```
import math

math.ceil(10.4)
11

math.ceil(-10.4)
-10
```
